# IDE Config Exports

OpenAgno publishes its documentation as a remote MCP server:

- MCP URL: `https://docs.openagno.com/mcp`
- LLM index: `https://docs.openagno.com/llms.txt`
- Full LLM context: `https://docs.openagno.com/llms-full.txt`

Use the files in this folder as copy-ready templates:

- `cursor-mcp.json`: copy to `.cursor/mcp.json` or `~/.cursor/mcp.json`
- `vscode-mcp.json`: copy to `.vscode/mcp.json`
- `windsurf-mcp.json`: merge into `~/.codeium/windsurf/mcp_config.json`
- `claude-code-setup.sh`: run from the project root to register the remote docs server in Claude Code
