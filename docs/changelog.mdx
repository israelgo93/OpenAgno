## v1.2.1

- closeout técnico del OSS después de la separación de repos
- metadata y versión pública alineadas con la organización OpenAgno
- dependencia base de Agno elevada a la línea `>=2.5.0`
- cleanup de lint/ruff en gateway, CLI, tools y tests
- validación reproducible de `pytest -q`, `python -m build`, `ruff check` y reinstalación local del wheel `1.2.1`

## v1.2.0

- **Published to PyPI** — `pip install openagno` now works globally
- tenant provisioning routes and tenant-aware agent runs
- Agno-native multi-tenant knowledge isolation using `linked_to` and `knowledge_filters`
- IDE config exports for Claude Code, Cursor, VS Code, and Windsurf
- bilingual onboarding updates for technical and non-technical users

## v1.1.0

- cierre técnico de Fase 9
- fallback top-level alineado con `workspace/config.yaml`
- CLI empaquetada `openagno`
- templates empaquetados
- rate limiting, tests y lockfile
- canales base para Slack y Telegram
- soporte actual para AG-UI y A2A

## v1.0.0

- estabilización del runtime AgentOS/FastAPI
- documentación Mintlify inicial
- supervisor local y comandos principales de operación
- sitio de documentación público con MCP activo

## v0.8.0

- endurecimiento del runtime
- auto-consciencia del agente con `workspace/self_knowledge.md`
