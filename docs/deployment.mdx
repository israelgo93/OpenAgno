## Modos soportados hoy

| Modo | Uso |
|------|-----|
| `openagno start --foreground` | desarrollo y debugging |
| `openagno start` | supervisor local |
| `systemd` | Linux con servicio persistente |
| `openagno deploy docker` | stack local con Docker Compose |

## Foreground

```bash
openagno start --foreground
```

## Supervisor local

```bash
openagno start
openagno status
openagno logs --follow
openagno stop
```

## systemd

```bash
sudo bash deploy/install-service.sh
sudo systemctl start openagno
sudo systemctl status openagno
journalctl -u openagno -f
```

## Docker Compose

```bash
docker compose up -d db
openagno deploy docker
docker compose --profile qr up -d
```

## Estado del despliegue AWS

La automatización AWS todavía no está implementada en la CLI. Sigue siendo parte del roadmap y no debe tratarse como capacidad cerrada.
