OpenAgno es una plataforma sobre **Agno** para construir y operar agentes con un **workspace declarativo**. El proyecto combina una CLI empaquetada, un runtime AgentOS/FastAPI, templates reutilizables, canales, knowledge con PgVector y soporte para servidores MCP.

## Lo que existe hoy

- CLI `openagno`
- workspace declarativo en YAML y Markdown
- runtime AgentOS con rutas operativas y knowledge
- templates empaquetados
- canales: WhatsApp, Slack y Telegram
- protocolos opcionales: AG-UI y A2A
- PgVector sobre PostgreSQL/Supabase
- documentación Mintlify y tests automatizados

## Lo que sigue en roadmap

- multi-tenancy productiva
- dashboard y auth
- remote execution y sandbox
- billing
- despliegue AWS automatizado

## Flujo recomendado

1. Instala el proyecto.
2. Crea un workspace desde un template.
3. Valida la configuración.
4. Arranca el runtime con `openagno`.
5. Conecta canales o clientes sobre el OS ya levantado.
