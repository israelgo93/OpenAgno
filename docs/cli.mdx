La ruta de operación documentada para OpenAgno es:

```bash
openagno <comando>
```

## Comandos principales

| Comando | Uso |
|---------|-----|
| `openagno init --template <id>` | Crear workspace desde template |
| `openagno templates list` | Listar templates disponibles |
| `openagno templates show <id>` | Ver metadata de un template |
| `openagno validate` | Validar `workspace/` |
| `openagno start --foreground` | Ejecutar el runtime en foreground |
| `openagno start` | Arrancar supervisor local |
| `openagno status` | Estado del runtime |
| `openagno logs --follow` | Seguir `gateway.log` |
| `openagno stop` | Detener supervisor |
| `openagno restart` | Reiniciar supervisor |

## Comandos de workspace

| Comando | Uso |
|---------|-----|
| `openagno create agent "<nombre>"` | Crear sub-agente |
| `openagno add whatsapp --mode <modo>` | Configurar WhatsApp |
| `openagno add slack` | Agregar Slack |
| `openagno add telegram` | Agregar Telegram |
| `openagno add agui` | Agregar AG-UI |
| `openagno add a2a` | Habilitar A2A |
| `openagno add tool <tool>` | Activar o desactivar tool opcional |
| `openagno deploy docker` | Levantar stack local con Docker Compose |
| `openagno deploy local` | Alias del supervisor local |

## Ejemplo completo

```bash
openagno init --template customer_support
openagno add slack
openagno add agui
openagno add tool tavily
openagno validate
openagno start --foreground
```
